# S.A.D.A.-v-1.5
SADA v1.5 — Open-source GUI for archaeological remote sensing. Load rasters, compute indices/PCA, run detectors (Standard, RX, Isolation Forest, LOF, OC-SVM) and clustering (K-means, DBSCAN), apply LISA, refine in the Anomaly Editor, and export GIS-ready rasters/shapefiles. Tooltips guide a human-in-the-loop workflow..
